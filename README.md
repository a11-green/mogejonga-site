# mogejonga-site
mogejonga on firebase用のリポジトリ

## URL
https://mogejonga-site.web.app/


## Deploy
GitHub上でPRを作成しmergeすると自動的に自動的にfirebaseにデプロイされる．
またPRを作成した時点でプレビュー用ページが作成される（URLは毎回異なる）．
